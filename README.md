# Attièkè Dékoungbé

Application web pour un restaurant africain.

## Structure
- `/client`: Frontend React + Vite + Tailwind CSS.
- `/server`: Backend Node.js + Express + SQLite.

## Démarrage Rapide

1. **Serveur**
   ```bash
   cd server
   npm install
   npm run seed
   npm start
   ```

2. **Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```

Ouvrez `http://localhost:5173` pour voir l'application.
Accédez à `http://localhost:5173/admin` pour le tableau de bord.
